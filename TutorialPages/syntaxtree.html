Greek and Hebrew Syntax Tool

A brief overview of using the Greek and Hebrew syntax tool.

This tutorial will guide you through using the Greek and Hebrew Syntax tools, and help you to better understand original Hebrew text.

Step-by-Step Guide

Step 1: Choose a Verse

When you first open the website, you should be introduced to a page that looks like this:

Depending on which language you chose you will see different options.

If you chose Hebrew you can only choose text from the Old Testament. The text under the logo should say Greek Syntax Search. Clicking on this text will take you to the Greek version of this tool.

If you chose Greek you will only be able to select verses from the New Testament. Just like the last section, there should be text under the logo to switch back to the Hebrew Syntax Search text.

If you would like to know more about the tool, hover over the yellow info icon next to the Syntax Search text.

Bible Verse Tree Tutorial

A brief overview of Syntactic Bible Tree app.

This tutorial will guide you through the different language functionalities used in the Syntactic Bible Tree app.

Access the website here.

This tool will provide great assistance with developing your language education in the bible as well as being able to dissect more complex sentences in the bible.

Step-by-Step Guide

Step 1: Getting Started

When you open the link to the site, you should notice a website, you should notice a navigation bar that looks like this

The "Book" option will let you select a verse in the bible to be viewed. The "Setting" option let you customize the output.

This will be explained further in the next step. Go ahead and select a verse from the "Book" option. You can only select one verse at a time.

Step 2: Adjusting Settings

Once you have selected a verse, you should see a tree structure like this pop up.

If you chose a verse from The Old Testament you should see a tree with Hebrew at the bottom. If you chose a New Testament verse, it should be in Greek.

When you hover over any of the blue text of the tree, you should see a small yellow box pop over the text. It looks like this:

This is the popover box that gives a reference number as well description of the part of speech that the word is used in. How to change the content of the popover is described below.

When you click the "Setting" button, you will see a popup window that looks like this

The Old Testament option determines how you want the Hebrew of the Old Testament to be displayed. Since Hebrew is read from left to right, you are given the option of reading it in its original direction.

The Glossary Language option lets you pick which language is displayed below the original text. You can choose English, Chinese, or both. An example of this can be seen in the Tree Example above, where the English below it is displayed below the Hebrew.

The Popover Language options lets you choose what language the popover box displays. Unlike Glossary Language you can only choose one language at a time.

Step 3: Understanding the Tree

Now that you have loaded the tree and understand the settings, you can begin analyzing the tree structure. Below is an explanation for the grammar rules and structure of the tree. These are applicable to both Greek and Hebrew translations.

For the purpose of this tutorial, here is another example tree, this time in Greek.

Top Label ("Verse"): Represents the entire sentence or verse being analyzed.

Tree Structure: the branches show how the sentence is broken down into smaller parts, called constituents.

Constituent Labels:

  • CL: Clause, the main part of the sentence containing a subject of the predicate.
  • SUBJ: Subject, the doer of the action or the topic of the sentence.
  • V: Verb, indicating the action or state of being.
  • PP: Prepositional Phrase, a modifying phrase consisting of a preposition and it's object.
  • OBJ: Object, which receives the action of the verb.
  • np: Noun Phrase, a group of words that act together as a noun.

Parts of Speech:

  • noun: Naming a word for a person, place, thing, or idea.
  • det:Determiner, a word that introduces a noun (like "the")
  • om: Object marker, in this case, it seems to be a symbol representing the absence of an object marker in the original text.
  • conj: Conjunction, a word that connects clauses or sentences or coordinates words in the same clause. (like"and)
  • verb: A word that represents an action or state of being
  • prep: Preposition, a word used to link nouns,pronouns, or phrases to other words within a sentence.

Direction of Analysis: The tree structure typically moves from top (the whole sentence) to bottom (individual words or morphemes), demonstrating how each part of the sentence relates to the whole.

Pericope Lookup Tutorial

A brief overview of the pericope lookup tool.

This tutorial will guide you through using the pericope tool.

A pericope is a set of verses that forms one coherent unit or thought, suitable for public reading from a text, now usually of sacred scripture.

Step-by-Step Guide

Step 1: Choosing a Version

When you first open the website, you should see a bar that looks like this at the top:

The default setting for this bar is the Chinese translation, however you can switch it to the NIV translation by selecting the dropdown menu next to the green toggle switch.

Step 2: Select a Verse

On that same navigation bar, is a small green book logo. Click this logo to open the verse selection tool and select a verse.

Regardless of what book you choose, the chapter of that verse will be opened, like so:

If you have the green toggle switch on, you will notice that the verse you selected will have a title above it, these are called Pericope Headings.

In the image above, the Pericope Heading is The Genealogy of Jesus the Messiah

Since pericopes can vary in length, the green highlighted text spans all verses within the current pericope. Click on any verse to see the associated pericope.

Pericope Lookup

Now that you have selected a verse, you should see a window pop up on the left side of the screen.

The Related pericopes section will return all the related pericopes throughout the bible. If you look above the right hand corner of the displayed pericopes, you should see a small grey slider.

This slider determines how many pericopes are displayed. By default, it will show all of them, but if you want to display ones closer to the current text, just slide it towards the right, and the number of displayed pericopes will decrease.

Do note that some verses don't have related Pericopes.
